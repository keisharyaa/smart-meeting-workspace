# Smart Meeting Workspace Application Skeleton

Copy the `src` and `docs` folders into the existing repository.

```powershell
Copy-Item ".\smart-meeting-workspace-application-skeleton\src" "." -Recurse -Force
Copy-Item ".\smart-meeting-workspace-application-skeleton\docs" "." -Recurse -Force
```

Then run:

```powershell
npm run lint
npm run build
git status
```
